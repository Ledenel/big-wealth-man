Detailed Description

代表调试用的日志，将调试信息与游戏界面字符分离。

可输出调试用的信息，并切换调试信息的输出模式。
例如，要检查TestCase中的某段代码是否能够运行，则可以在代码中加入如下语句：
Logger.log("TestCase","Something done.");
然后通过观察日志输出来确认代码是否成功运行。
