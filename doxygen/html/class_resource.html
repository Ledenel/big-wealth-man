Detailed Description

代表一个游戏资源管理器.

所有的游戏资源（对话，文字，地图显示及提示信息等）都可以从该管理器中获得。
通过指定的资源名来获取文本及颜色资源。
资源名使用.来分隔，以唯一标识资源。
例如：
"player.A.name" 代表A玩家的名称.
"player.panel" 代表玩家的信息面板.
"cell.bank.display" 代表游戏地图上的银行的显示样式.
"cell.bank.view" 代表银行的操作界面.
